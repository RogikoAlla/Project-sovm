ОПИСАНИЕ
────────
Карточная игра для 4 игроков по сети, основанная на классическом
«Подкидном дураке» с одним отличием: Король (ранг) сильнее Туза.

Цель игры — оставаться Королём как можно дольше, пока остальные
игроки пытаются занять это место.

УСТАНОВКА
─────────
1) Распакуйте архив:
   unzip KAS.zip
   cd KAS

2) Создайте виртуальное окружение (рекомендуется):
   python3.12 -m venv .venv
   source .venv/bin/activate        # Linux / macOS
   .venv\Scripts\activate           # Windows

3) Установите проект со всеми зависимостями:
   pip install -e ".[dev]"

   Зависимости для запуска: только стандартная библиотека + babel.
   Зависимости для разработки: pytest, flake8, pydocstyle, sphinx, doit.


ЗАПУСК ИГРЫ
───────────
Откройте 5 терминалов (1 сервер + 4 клиента).

  Терминал 1 — сервер:
    kas-server                        # колода 36 карт (по умолчанию)
    kas-server --deck 52              # колода 52 карты
    kas-server --host 0.0.0.0         # доступен из сети
    kas-server --port 65432 -v        # с подробными логами

  Терминалы 2–5 — клиенты:
    kas-client --name Иван
    kas-client --name Маша
    kas-client --name Петя
    kas-client --name Оля

  Игра запускается автоматически, как только подключатся 4 игрока.

  Если игроки на разных компьютерах:
    # Узнайте IP сервера: hostname -I
    kas-client --host 192.168.1.XXX --name Иван


ПРАВИЛА ИГРЫ
────────────
1. РАЗДАЧА
   36 карт раздаётся по 9 каждому (или 52 по 13).
   У кого оказался Король Пик — тот становится Королём.
   Роли присваиваются против часовой стрелки:
     Король → Туз → Дама → Слуга

2. НАЧАЛО РАУНДА
   После раздачи даётся 10 секунд, чтобы рассмотреть карты и роли.
   Затем Королю предлагается слепой обмен руками, потом выбор козыря.

3. СЛЕПОЙ ОБМЕН (опционально, 1 раз за игру)
   Король может поменять свои карты с любым другим игроком вслепую
   (не видя его руку). После обмена все видят, с кем поменялся Король.

4. КОЗЫРЬ
   Король объявляет козырь из мастей, которые есть у него в руке.
   Все игроки видят объявленный козырь.

5. ХОД ИГРЫ (правила Подкидного дурака + Король > Туз)
   Ходят по часовой стрелке:
     Слуга атакует Даму
     Дама атакует Туза
     Туз атакует Короля
     Король атакует Слугу
     … и так по кругу

   АТАКА:
   • Первый ход — карты одного ранга (одна или несколько).
   • Подкидывание — только карты тех рангов, что уже есть на столе.
   • Максимум 6 карт в одной атаке.
   • Нельзя подкинуть больше карт, чем есть у защитника в руке.

   ЗАЩИТА:
   • Бьёт картой той же масти, но старшей — или любым козырем.
   • Козырь бьётся только козырем старше.
   • Король (ранг) сильнее Туза во всех мастях.
   • Если защитник не смог или не хочет отбиться — забирает карты.

   БИТО:
   • Объявляет только атакующий.
   • После того как все атакующие карты отбиты, атакующий может
     подкинуть ещё (разрешённые ранги) или сказать «бито».
   • При «бито» карты уходят в отбой, ход переходит к следующей паре.

   СМЕНА РОЛЕЙ (правило 5):
   • Если защитник взял карты — атакующий и защитник меняются ролями
     (исключение: Слуга, взявший у Короля, не меняется ролями).

6. КОГДА ЗАКАНЧИВАЮТСЯ КАРТЫ
   Игрок, у которого закончились карты, сохраняет свою роль до конца
   текущего раунда.

7. НОВЫЙ РАУНД
   Если в следующей раздаче кому-то выпадет Король Пик снова — он НЕ
   становится Королём автоматически. Роли переходят из прошлого раунда.


УПРАВЛЕНИЕ В ИГРЕ
─────────────────
Слепой обмен:
  [1], [2], [3] — выбрать игрока для обмена
  [0]           — отказаться от обмена

Выбор козыря:
  [1]–[4]       — выбрать масть из списка (показаны только масти из вашей руки)

Атака:
  1 3 5         — сыграть карты [1], [3] и [5] из вашей руки
  0             — пас (первый ход) / бито (подкидывание)

  Пример: у вас в руке 9 карт, вы хотите атаковать картами №2 и №2 (одного ранга):
    > 2 5

Защита:
  1-3 2-7       — бить карту стола [1] картой руки [3],
                  и карту стола [2] картой руки [7]
  0             — забрать все карты

  Пример: на столе 2 карты [1] и [2], у вас карта №4 бьёт [1], карта №6 бьёт [2]:
    > 1-4 2-6

Таймеры:
  У каждого хода есть 28 секунд на ввод.
  Остальные игроки видят обратный отсчёт.
  При истечении времени: атака → пас, защита → забрать, козырь → авто.

ИЗВЕСТНЫЕ ОГРАНИЧЕНИЯ
─────────────────────
• Игра только по локальной сети (TCP). Публичный интернет — через
  туннель (ngrok, tailscale и т.п.) или проброс портов.
• Ровно 4 игрока, не больше и не меньше.
• Переподключение после разрыва не поддерживается.
• Логика смены ролей после взятия (правило 5) влияет на турнирные роли,
  но немного упрощена для читаемости кода.
